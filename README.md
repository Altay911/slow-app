# slow.

> In a world obsessed with speed, slow. gives you permission to breathe.

A mobile app for night people. A grounding tool — not a motivation app. Just a quiet space to remember what actually matters.

## Tech Stack

- **Backend:** FastAPI + Python
- **Database:** PostgreSQL 17
- **ORM:** SQLAlchemy 2.0 async + Alembic
- **Infra:** Docker, AWS, GitHub Actions CI/CD
- **AI:** GCP Vertex AI
- **Mobile:** React Native
